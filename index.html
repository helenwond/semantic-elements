Semantic Elements

Semantic elements are elements which describe the content of the page and should also give meaningfor the developers. When html was first invented, there was no CSS tag so that the developers need to add more tags to format the pages, but this made the page more sophisticated and was not easy to maintain. Then they invented CSS which will help to organize the format of a page. But the CSS by itself doesnâ€™t explain itself since we can override the tags change the meaning of those tags.

The HTML 5 semantic elements are self-explanatory, meaningful, readable and compatible for any browser. The strong(<strong>) and emphasis(<em>) tags are two examples of semantic tags.
